包名：com.example.monopoly_deal_game.view
================================================================================
【包含的类】
  - GameObserver  会话更新时通知 UI（观察者模式）。
  （另有 package-info.java。）

【接口】
  - GameObserver：onSessionUpdated(GameSession)。

【需要实现什么】
  - 按需求 17 拆分自定义控件：卡牌 Tooltip、玩家资产缩略区、中央手牌区等。
  - 可在此包实现 GameObserver 的具体类，或在 controller 中匿名实现。
  - 与 JavaFX Property 绑定二选一：观察者或双向绑定只读 VM。

【与其它包的关系】
  - → model：只读 GameSession / DTO，不直接改模型。
  - ← logic：引擎状态变更后触发 onSessionUpdated（由 controller 或引擎回调注册）。
  - controller：FXML 的 fx:controller 仍放在 controller 包；本包放可复用 Node/控件。
